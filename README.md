# SMTP Email Communication System

## Overview
This project demonstrates how two email clients can communicate using the SMTP protocol.

## Features
- Email sending between two accounts
- SMTP protocol implementation
- Basic email message structure handling

## Technologies
- Python
- SMTP protocol
- Email libraries

## How It Works
1. Client connects to SMTP server
2. Message is formatted
3. Email is sent to recipient account

## How to Run
1. Configure email credentials
2. Run Python script
3. Send test message

- SMTP protocol fundamentals
